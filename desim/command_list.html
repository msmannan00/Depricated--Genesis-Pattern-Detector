open_classifier

This would open a software to train models to classify objects

extract_code

To extract the uploaded zip file of your project. Would also clear the cache if you want to start all over again

file_support|$type

All file types are set by default but if you want to specify file types use the following command like file_support|java

regex|$str

If you want to find pattern in project use regex command as shown above like regex|a+

compile

Compile and index all files of given software according to required settings

clear_all

This code would clear all files and history

clear_history

This code would clear the command history

init_parser|$type

This could would initialize any parser currently installed and configured in system
